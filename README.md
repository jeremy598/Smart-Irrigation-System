# Smart Irrigation System

This project presents the design and implementation of a smart irrigation system
using a microcontroller, soil moisture sensor, relay module, and water pump.

## Features
- Automatic irrigation based on soil moisture level
- Microcontroller-based real-time control
- PCB designed using KiCad
- Gerber files generated for fabrication

## Tools Used
- KiCad (Schematic & PCB Design)
- Arduino (Microcontroller Programming)
